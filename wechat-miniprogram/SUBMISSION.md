# 微信小程序递交前清单

## 必填配置

- 将 `wechat-miniprogram/project.config.json` 里的 `appid` 从 `touristappid` 替换为真实小程序 AppID。
- 使用管理员或开发者微信扫码登录微信开发者工具。
- 在微信公众平台完成“小程序隐私保护指引”配置。
- 如果语音概要继续使用线上地址，需要把 `https://newlife30.com` 加入合法域名。

## 本地检查

```bash
npm run miniprogram:check
npm run miniprogram:submit-check
npm run build
```

`miniprogram:submit-check` 会检查页面文件、Tab 图标、HTTPS 地址、课程音频和主包体积。它会把 AppID 和域名校验作为 warning 提醒，因为这些依赖正式账号后台配置。

## 体验版测试路径

1. 今日：校准开始日期，选择日期，勾选任务，填写输出和复盘，完成今日打卡。
2. 打卡：切换每日打卡和课程作业，填写内容，复制群发文案。
3. 系统：切换 8 个个人运行系统，确认核心内容、价值和训练方式完整。
4. 课程：勾选 4 次核心课，切换 8 个系统课程，播放/暂停语音概要。
5. 小组：标记满勤、补打卡，填写组长后台记录，复制统计文案。
6. 平台说明：确认平台定位、数据隐私和健康内容边界表达准确。

## 递交说明建议

小程序简介可写：

```text
30天重建人生体系是一款课程学习与自我打卡工具，帮助用户围绕身心能量、节律、目标、反馈、学习、关系、价值和身份八个系统，完成30天日课记录、课程学习、小组复盘和积分统计。
```

类目建议优先选择教育、效率或生活服务相关类目，避免使用医疗诊疗、心理治疗等承诺性表达。
